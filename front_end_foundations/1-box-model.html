The Box Model

It's actually important to reset the default browser styles before making modifications so that I know how my code is affecting the result.

html, body, hl, h2, h3, p, ol, ul, li, a {
    padding: 0;
    border: 0;
    margin: 0;
}

Then start with the body tag and put padding around all sides:

body {
	padding: 20px 20px 20px 20px;
}

...or use <div>'s for sections to do the same thing. Then focus on the heading tags:

h1 {
	margin: 10px 0 15px 0;
}

We can use border to create a horizontal line like the <hr> tag. 

h3 {
	margin: 15px 0 15px 0;
	border-bottom: 1px solid #CCCCCC;
	padding-bottom: 3px;
}

Also, to indent lists:

ul {
	padding: 0 0 0 50px;
}

ol {
	padding: 0 0 0 50px;
}
