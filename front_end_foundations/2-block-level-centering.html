Block Level & Centering

Block Level

img is an "inline-level" tag while a text tag is a "block-level" tag.

To make something like an image "block-level" so it can be centered:

.header img {
    display: block;
}

Center

There are two main ways to center things, but the CSS you write depends on what you're centering.

  • If you want to center an entire block-level tag, and it is fixed-width, then set the left and right margins to "auto."
  • If you want to center content (children) that are inside a block-level tag, then use:
text-align: center;
