Images & Background Images

There are three main types of webpage images:

  • Content images: just as necessary to the page as all of the text.

  • Layout images: These are in the background, and are not necessary to understand the content of the page.

  • User interface images: These assist with the interface of the webpage, but are not required to understand the content of the page.

Background images aren't set in the HTML document. They're set in CSS.

There are several different properties that relate to background images. background-color should always be set in case the background image fails to load.

body {
    background-color: #5f9ea0;
    background-image: url(assets/code-school-logo.png);
}

background-repeat makes an image repeat, or not, on the background. Values can be "repeat" (which is the default if not specified), "repeat-x" (tiles images horizontally), "repeat-y" (tiles images vertically), and "no-repeat."

body {
    background-color: #5f9ea0;
    backgroung-image: url(assets/code-school-logo.png);
    background-repeat: no-repeat;
}

background-position can be used to position the image in a container and uses two keywords.

The first keyword can be either "top," "center," or "bottom."

The second keyword can be either "left," "center," or "right."

body {
    background-color: #5f9ea0;
    background-image: url(assets/code-school-logo.png);
    background-repeat: no-repeat;
    background-position: bottom right;
}

background can also be defined in one line by first defining the color, then the image url, then the position, then the repeat.

body {
    background: #5f9ea0 url(assets/code-school-logo.png) top center
    no-repeat;
}

We can set different backgrounds for different containers:

body {
    background: #000000 url(images/stardust.png) top left repeat;
}
.main-content {
    background: #ffffff url(images/crossword.png) top left repeat;
}
.recipes li {
    background: #ffffff url(images/li-bg.png) top left repeat;
}

We can create an empty div container specifically for holding a background image, and put text (HTML) tags inside of it.

.foundations-empty-div-example {
    background-color: #5f9ea0;
    background-image: url(assets/code-school-logo.png);
}
.foundations-empty-div-example p {
    font-family: Impact, Haettenschweiler, 'Arial Narrow Bold',
        sans-serif;
    font-size: 40px;
    line-height: 0;
    text-align: center; 
}

<div class="foundations-empty-div-example"><p>Some random text placed
over the background.</p></div>
