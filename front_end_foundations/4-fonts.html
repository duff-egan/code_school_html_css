Use quotes if fonts are multi-word:

body {
    font-family: Helvetica, Arial, "Times New Roman", sans-serif;
}

We can reset all of the font properties to a baseline default so that the browser isn't doing it anymore.

html, body, hl, h2, h3, p, ol, ul, li, a {
    padding: 0;
    border: 0;
    margin: 0;
    font-size: 100%;
    font: inherit;
}
font-weight: bold;

"normal" is the default weight. We can also use:

  • lighter
  • bolder
  • 100, 200 ... 900
font-style: italic;
text-transform: uppercase;
font-weight: bolder;
