Border Radius

The "border-radius" property allows us to apply rounded corners to borders.

We could do this by specifying four different "border-radius" properties:

.box {
    background: grey;
    height: 50px;
    width: 200px;
    border-top-left-radius: 15px;
    border-top-right-radius: 15px;
    border-bottom-right-radius: 15px;
    border-bottom-left-radius: 15px;
}

We can use the shorthand border-radius property to specify all corners at once:

.box {
    background: grey;
    height: 50px;
    width: 200px;
    border-radius: 15px;
}

We can also use shorthand to specify each border-radius value individually in order:

border-radius: <top left> <top right> <bottom right> <bottom left>
.box {
    background: grey;
    height: 50px;
    width: 200px;
    border-radius: 4px 30px 12px 20px;
}

    height: 50px;
    width: 200px;

Box Shadow

The "box-shadow" property allows us to apply a shadow on a given element.

The box-shadow property has several value inputs:

box-shadow: <inset> <offset-x> <offset-y> <blur-radius>
    <spread-radius> <color>
  • If "inset" is not specified (which is the default), a drop shadow is created. If you use the keyword "inset" then an inset shadow is created.
  • The "offset-x" value moves the shadow a certain distance on the x-axis, or left and right.
  • The "offset-y" value moves the shadow a certain distance on the y-axis, or up and down.
  • The "blur-radius" value alters the amount of blur on the shadow. Larger values cause it to become bigger and lighter.
  • The "spread-radius" causes the shadow to exand or shrink.
  • The "color" value allows us to change the color of the shadow.

The following code doesn't specify an inset shadow, so it's going to be a drop shadow. Then we're specifying a 3px offset-x value, a 3px offset-y value, and a 4px blur-radius. We're not specifying a spread-radius value, so the drop shadow is the same size as the element. And, in this case we're specifying a blue color for the shadow:

.box {
    background: grey;
    height: 50px;
    width: 200px;
    box-shadow: 3px 3px 4px #00f;
}
