Overview

HTML5 Overview

HTML5 is the new standard, which includes:

  • New HTML elements and attributes
  • Full CSS3 support
  • Video and audio elements
  • 2D/3D graphics
  • Local Storage
  • Local SQL database

CSS3 Overview

CSS3 is the latest standard of CSS:

  • Border radius
  • Box shadow, text shadow
  • Transitions and transforms
  • Gradients
  • Multiple backgrounds
  • Font face

Note: text shadow and font face are not new features, but they're still important components of modern CSS, so they'll be covered here.

Updated HMTL5 Elements

In HTML5, the doctype has been simplified from previous versions:

<!DOCTYPE html>

In HTML5, the meta declaration has been simplified from previous versions:

<meta charset="UTF-8">

In HTML5, the script tag has been simplified from previous versions and no longer requires the type attribute:

<script src="file.js"></script>

In HTML5, the link tag has been simplified from previous versions and no longer requires the type attribute:

<link rel="stylesheet" href="file.css">

Existing HTML5 Tag Updates

Traditionally presentational tags, the i, b, em, and strong tags have been given new semantic meanings.

In HTML4, the i and b tags were font styles elements:

  • The i tag rendered an italic font style.
  • The b tag rendered a bold font style.

In HTML5, these tags have new meanings:

  • The i tag represents text in an "alternate voice" or "mood." For example:
    • Taxonomic designation
    • Technical term
    • Idiomatic phrase from another language
    • Transliteration
    • A thought
    • Ship name in Western texts

Example use of the i tag:

<p><i>I hope this works</i>, he thought.</p>

I hope this works, he thought.

  • The b tag represents "stylistically offset" text. For example:
    • Key words in a document abstract
    • Product names in a review
    • Actionable words in interactive text-driven software
    • Article lead

Example use of the b tag:

<p><b class="lead">The event takes place this upcoming Saturday, and
    over 3,000 people have already registered.</b></p>

The event takes place this upcoming Saturday, and over 3,000 people have already registered.

In HTML4:

  • The em tag meant "emphasis."
  • The strong tag meant "strong emphasis."

In HTML5, these tags have new meanings:

The em tag now means "stress emphasis."

The strong tag now means "strong importance." For example:

Example use of the em and strong tags. Here we're giving "stress emphasis" to the word "before," and "strong importance" to the date of the event:

<p>Make sure to sign up <em>before</em> the day of the event,
    <strong>September 16, 2013</strong>.</p>

Make sure to sign up before the day of the event, September 16, 2013.
